üß™ How to Use TabloForge?

1. Prepare Your JSON

Create a JSON file with a rows array containing your table data. Example: 

{
  "rows": [
    [{"text": "Header 1", "color": "#4c1"}, {"text": "Header 2", "color": "#4c1"}],
    [{"text": "Data 1", "icon": "success", "color": "transparent"}, {"text": "Data 2", "icon": "check", "color": "transparent"}]
  ]
}
2. Upload and Use

Upload your JSON to a public URL (e.g., GitHub, Vercel), then use it in the TabloForge URL: 

https://tablo-forge.vercel.app/api/table?json=https://your-site.com/my-table.json&_canvas=600x200&_theme=dark
3. Use Query Parameters

Create a table directly with query parameters like r1=ID,Name,Score. Example: 

https://tablo-forge.vercel.app/api/table?r1=ID,Name,Score&r2=1,Alice,95&r3=2,Bob,88&_canvas=600x200&_theme=dark
